# MyCollegeProjects / Мои проекты с колледжа
There is a few projects that i made (with a help of ai of course) for my course projects in college. You may not be interested in them because they only have russian language.
Здесь хранится пара проектов сделанных мной (сделанных с помощью нейросети конечно) для курсовых проектов в колледж.

The "Сайт доставки японской еды" is a website for japanese restaurant, where you can order some food (doesn't send any messages btw).
"Сайт доставки японской еды" это веб сайт для японской кафешки где можно заказать доставку еды (не отправляет сообщений никуда кстати).

The "RaschetZP" is a emloyee wage calculator with a little database and todo list.
"RaschetZP" является калькулятором сотрудников заработной платы с небольшой базой данных и списком задач.
